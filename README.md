# Kaggle-TensorFlow-Titanic
Kaggle - Titanic: Machine Learning from Disaster

Competition Description
-------
The sinking of the RMS Titanic is one of the most infamous shipwrecks in history. In this challenge, we ask you to complete the analysis of what sorts of people were likely to survive. In particular, we ask you to apply the tools of machine learning to predict which passengers survived the tragedy.
More details please visit https://www.kaggle.com/c/titanic

Code Environment
-------
You can use [Anaconda](https://www.anaconda.com/download/) to install the code environment.
1. conda env create -f environment.yaml
2. source activate tensorflow
3. jupyter notebook

Implementation Details
-------
1. Data processing.
2. Data normalization.
3. Data split into train set, dev(cross validation) set, test set.
4. Data classification by using Deep Learning implemented in TensorFlow.
